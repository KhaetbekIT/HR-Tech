## Запуск проекта HarmonyHR

`для проверки последний версий NodeJS 20.х`
```bash
node -v 
```
`для установки node_modules`
```bash
npm install 
```
`запуск проект на` [http://localhost:3000](http://localhost:3000/HR-Tech)
```bash
npm run dev
```
